# DALi-SoundKit
A linux based Digital Audio Lab

The DALi-SoundKit is a collection of applications and libraries written in C that together form a virtual music/sound production environment for linux. It aims to have as little dependencies as possible with the exception of some core API's like GLFW for window/input management or ALSA for interacting with audio hardware.
